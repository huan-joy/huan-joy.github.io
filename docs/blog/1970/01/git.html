    # Git GUI的汉化

    Git bash的汉化：在命令行界面点击右键-选择Options-Windows-然后在里面将language设置为zh_CN。 Git GUI：

    1. 打开https://github.com/stayor/git-gui-zh (opens new window)
    2. 获取目录下【zh_cn.msg】文件
    3. 在git的安装目录下找到【C:\Program Files\Git\mingw64\share\git-gui\lib】
    4. 在【lib】下创建个文件夹【msgs】，将【zh_cn.msg】文件放入
    5. 重启Git Gui即可
